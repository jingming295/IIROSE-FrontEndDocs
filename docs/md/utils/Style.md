# Style

## Utils.Style.fontSize(e, t)

> 设置聊天室字体大小

```javascript
Utils.Style.fontSize(e, t)
```

```e```: 当 ```e``` 为 0 时，表示公共聊天消息的字体大小需要被设置。

```t```: 字体大小，```t```的值加上11


### 设置聊天室字体大小

- 使用例子

```javascript
Utils.Style.fontSize(0, '20')
```
