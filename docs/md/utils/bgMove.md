# bgMove

## Utils.bgMove(e,t)

> 开启或关闭移动壁纸功能

```javascript
Utils.bgMove(e,t)
```

```e```: 布尔值，true为开启，false为关闭

```t```: 布尔值，true为不通知用户，false或者不提供为会通知用户

### 开启或关闭移动壁纸功能

- 使用例子

```javascript
Utils.bgMove(true, false)
```
