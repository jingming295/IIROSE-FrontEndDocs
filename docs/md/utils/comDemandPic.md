# comDemandPic

## Utils.comDemandPic(e,t,o,i,a)

> 根据不同的选项拼接文本

```javascript
Utils.comDemandPic(e,t,o,i)
```

```e```: 布尔值，如果为真，就不执行任何操作

```t```: 数字，有0，1，2，3，4的选项

```o```: 需要拼接的文本

```i```: 拼接的时候需要用到的字符串

### 根据不同的选项拼接文本

- 使用例子

```javascript
var result = Utils.comDemandPic(false, 0, "http://r.iirose.com/i/20/6/9/19/0934-NJ.gif#e", 100);
console.log(result);
```
