# danmakuMode

## Utils.danmakuMode(e,t)

> 切换弹幕的显示模式

```javascript
Utils.danmakuMode(e,t)
```

```e```: 数字，为要切换的弹幕模式的选项

|```e```|对应选项|
|:--:|--|
|0|完整气泡|
|1|轻量气泡|
|2|静默接收|
|3|拒收|
|01|跟随睡眠模式  :  完整气泡  >  轻量气泡|
|02|跟随睡眠模式  :  完整气泡  >  静默接收|
|03|跟随睡眠模式  :  完整气泡  >  拒收|
|12|跟随睡眠模式  :  轻量气泡  >  静默接收|
|13|跟随睡眠模式  :  轻量气泡  >  拒收|
|23|跟随睡眠模式  :  静默接收  >  拒收|

```t```: 为真的时候，不通知用户已经切换

### 切换弹幕模式

- 使用例子

```javascript
Utils.danmakuMode(0, false)
```
