# imgReady()

> 图片相关

## Utils.imgReady.set(e,t)

```javascript
Utils.imgReady.set(e,t)
```

```e```: HTMLImageElement
```t```: 想要设置的图片的url

### 设置图片

```javascript
const albumShow = document.getElementById(`albumShow`)
Utils.imgReady.set(albumShow, `https://static.codemao.cn/i/24/5/6/13/0001-W7.png`)
```

- 效果：

![设置图片](https://static.codemao.cn/i/24/5/5/19/2658-9U.png)
